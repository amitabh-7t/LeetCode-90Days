# LeetCode 90 Days Challenge

This repository contains my solutions for the LeetCode 90 Days challenge.

## Structure

The solutions are named using the following convention:
`Q<DayNumber>-<LeetCodeID>-<ProblemName>.py`

For example:
- `Q1-1-Two Sum.py`: Solution for Day 1, LeetCode Problem 1 (Two Sum).
